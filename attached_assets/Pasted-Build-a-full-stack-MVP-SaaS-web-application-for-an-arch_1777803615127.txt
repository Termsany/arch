Build a full-stack MVP SaaS web application for an architecture and interior design company.

The app should be in Arabic RTL and should help architecture/interior design offices manage client projects through the full design and execution workflow.

Use this stack:
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MySQL
- Styling: simple clean CSS, Arabic RTL layout
- Authentication: simple JWT login
- Project structure:
  - frontend/
  - backend/
  - database/

The system has two main parts:

Part 1:
Internal project management for architecture/interior design work:
- Clients
- Projects
- Project workflow stages
- Client approvals and revisions
- Estimates / BOQ

Part 2:
SaaS admin management:
- Subscription plans
- Office accounts preparation
- Pricing page

Important:
This is an MVP.
Keep it simple, clean, and working.
Do not add payment integration yet.
Do not overcomplicate permissions yet.
Focus on working CRUD, workflow, plans management, and Arabic RTL UI.

- cancelled

Database table:

CREATE TABLE offices (
  id INT AUTO_INCREMENT PRIMARY KEY,
  office_name VARCHAR(150) NOT NULL,
  owner_name VARCHAR(150),
  phone VARCHAR(50),
  email VARCHAR(150),
  address TEXT,
  plan_id INT,
  subscription_status ENUM('trial', 'active', 'expired', 'cancelled') DEFAULT 'trial',
  subscription_start DATE,
  subscription_end DATE,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  FOREIGN KEY (plan_id) REFERENCES subscription_plans(id) ON DELETE SET NULL
);
